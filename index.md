# Code Learning Log

Welcome to my personal code learning log.

This site records my learning process in programming, especially in Python and C++. I use this place to document the coding problems I encounter, the solutions I try, and the lessons I learn during self-study.

## What I Record

* Python practice
* C++ practice
* Algorithm exercises
* Debugging notes
* Small coding projects
* Reflections on mistakes and solutions

## Why I Created This Site

I created this site not only to store code, but also to record the process of learning programming.

When I solve a problem, I want to document:

* what the problem was
* how I understood it
* what mistakes I made
* how I fixed them
* what I learned from the experience

## Learning Topics

### Python

I will use this section to record Python basics, small exercises, and problem-solving examples.

### C++

I will use this section to record C++ syntax, object-oriented programming, pointers, memory-related concepts, and practice code.

### Notes

I will also write notes about common errors, debugging experience, and useful programming concepts.

## Goal

The goal of this site is to build a clear and organized record of my coding journey, so that I can review my progress and share my learning experience with others.
